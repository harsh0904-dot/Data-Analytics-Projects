# SQL Practice

This folder is designed to demonstrate SQL depth beyond project-specific dashboards.

## Purpose

- Strengthen query writing across beginner, intermediate, and advanced topics.
- Build interview-ready business problem solving using SQL.
- Showcase structured learning progression for recruiters and hiring managers.

## Folder Overview

- `01_Fundamentals`: core filtering, sorting, grouping, and aggregation.
- `02_Intermediate_Queries`: joins, subqueries, CTEs, and conditional logic.
- `03_Advanced_Analytics`: window functions, ranking, cohort logic, and KPI analysis.
- `04_Business_Case_Questions`: realistic business questions solved with SQL.

## How To Use

- Practice one topic at a time.
- Add dataset assumptions at the top of each SQL file.
- Keep queries business-focused and well commented.
- Include expected outputs for important questions.
