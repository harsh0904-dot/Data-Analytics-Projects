# Dashboard Templates

This folder contains reusable Power BI dashboard concepts for fast portfolio project setup.

## Purpose

- Reuse strong dashboard layouts across industries.
- Maintain consistent design quality across projects.
- Speed up future report building with repeatable structure.

## Template Categories

- `01_Executive_KPI_Template`: high-level leadership reporting.
- `02_Sales_Performance_Template`: revenue, orders, targets, and trends.
- `03_Finance_Dashboard_Template`: profitability, cost, budget, and variance.
- `04_HR_Dashboard_Template`: workforce, attrition, diversity, and engagement.
- `05_Healthcare_Dashboard_Template`: patient, outcome, cost, and operations metrics.

## How To Use

- Start with a template before building a new dashboard.
- Adapt KPIs and color palette to the industry.
- Keep navigation, slicers, and card layout consistent.
