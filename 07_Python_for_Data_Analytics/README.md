# Python for Data Analytics

This folder demonstrates practical Python workflows used in analytics projects.

## Purpose

- Show clean, reusable analytics scripting patterns.
- Build confidence in data cleaning, EDA, feature engineering, and automation.
- Support portfolio storytelling with modular Python examples.

## Folder Overview

- `01_Data_Cleaning`: missing values, type conversion, duplicates, standardization.
- `02_EDA_and_Visualization`: descriptive profiling and quick visual summaries.
- `03_Feature_Engineering`: business-ready derived variables.
- `04_Automation_and_Utilities`: repeatable reporting and export helpers.
- `05_Mini_Projects`: compact end-to-end Python exercises.

## Best Practice Focus

- Use clear function-based structure.
- Separate raw, cleaned, and output paths.
- Print quality checks after every transformation.
- Save outputs in analyst-friendly formats such as CSV or Excel.
