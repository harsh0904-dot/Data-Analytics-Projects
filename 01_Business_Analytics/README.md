# Business Analytics

This section contains business-focused analytics work built for executive decision-making.

## Project Included

### Executive Business Dashboard

An end-to-end analytics project that translates business requirements into KPI-driven analysis and dashboard storytelling.

## Objective

Build an executive-ready dashboard that answers high-priority business questions across sales, profitability, customer behavior, product performance, and regional trends.

## Business Scope

The project is designed to support decisions such as:
1. How the business is performing overall (sales, profit, orders, customers, margin)
2. Which regions, states, and cities are driving or reducing performance
3. Which product categories and sub-categories generate revenue and profit
4. Which customer segments and accounts contribute the most value
5. Where discounting and profit leakage require immediate action

## Tools and Tech Stack

1. SQL: KPI queries and business analysis
2. Python (Pandas): data cleaning and preprocessing
3. Power BI: executive dashboard and visual storytelling
4. Markdown Documentation: BRD, KPI definitions, business questions, and roadmap

## Project Workflow

1. Business Requirement Definition (BRD)
2. Business Question Catalog
3. KPI Planning and Metric Definitions
4. Data Understanding and Validation
5. Python Data Cleaning
6. SQL Analysis and KPI Querying
7. Power BI Dashboard Development
8. Insight Generation and Recommendations
9. Portfolio Documentation and Version Control

## Core KPIs Covered

1. Total Sales
2. Total Profit
3. Total Orders
4. Total Customers
5. Profit Margin %
6. Average Order Value
7. Average Discount %
8. Average Profit per Order
9. Total Quantity Sold
10. Number of Products Sold
