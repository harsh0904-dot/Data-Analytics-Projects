# Project Approach

## 1) Data Preparation

- Load raw IBM HR attrition dataset.
- Standardize column names and categorical values.
- Remove duplicates and handle missing values.
- Export cleaned dataset to Excel for BI consumption.

## 2) SQL Analysis

- Build KPI queries for headcount, attrition, and average metrics.
- Build analysis queries for department, role, travel, and education segmentation.
- Use grouped and percentage logic for chart-ready outputs.

## 3) Dashboard Development (Power BI)

- Build executive summary page with KPI cards.
- Add attrition analysis visuals (stacked bars, funnel, waterfall, donut).
- Add workforce performance visuals (income by level, department comparison table).
- Add slicers for Department, JobRole, BusinessTravel, and Attrition.

## 4) Validation and Documentation

- Cross-check KPI totals between SQL and Power BI.
- Validate screenshot values (1.47K total, 237 exits, 16.12% attrition).
- Document business questions, data model, DAX, and insights.
