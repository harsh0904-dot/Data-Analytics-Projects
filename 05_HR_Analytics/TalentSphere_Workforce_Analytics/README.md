# TalentSphere Workforce Analytics

- Domain: HR Analytics (Employee Attrition and Workforce Performance)
- Tool stack: MySQL, Python (Pandas), Power BI
- Source dataset: IBM HR Analytics Employee Attrition

## Project Goal

- Identify drivers of attrition and highlight workforce trends for HR leadership.
- Build an executive-ready dashboard for fast decision making.

## Folder Map

- `01_Dataset`: Raw and cleaned HR datasets.
- `02_SQL`: KPI and analysis queries.
- `03_Python`: Cleaning and EDA scripts.
- `04_Power_BI`: Dashboard notes and build steps.
- `05_Documentation`: BRD, model, DAX, insights.
- `06_Screenshots`: Final dashboard page images.

## Dashboard Pages

- HR Executive Analytics
- Employee Attrition Analysis
- Workforce Performance

## Key Headline Metrics

- Total Employees: 1.47K
- Active Employees: 1.233K
- Employees Left: 237
- Attrition Rate: 16.12%
- Average Monthly Income: 6.50K
- Average Age: 36.92

## How To Use

- Run SQL in `02_SQL` to validate KPIs and segment analysis.
- Use cleaned dataset in Power BI.
- Rebuild visuals using notes in `04_Power_BI`.
- Review business context and findings in `05_Documentation`.
